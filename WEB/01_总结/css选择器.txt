1.标签名选择器：通过标签名选择多个该标签
2.类选择器：通过class属性名查找，选择多个该属性名的标签
		一个页面中每个class的值不唯一
		一个标签的class属性值可以为多个
3.id选择器：通过id属性值，选择单个该属性标签
		每个id都是唯一的
		id属性的名字只可以有一个
4.后代选择器：通过父级查找其子级元素，中间用空格连接
		扩展：如果想选择该父级的直系子级元素中间用>连接
5.群组选择器：把具有相同重复样式的选择器放在一起，节省代码量，用逗号（“，”）隔开
6.组合选择器：通过两种组合器的组合，达到精确查找的目的
7.全局选择器（通配符选择器）：选择页面中所有标签。
	经常用来进行标签样式的初始化。
		*{
			margin: 0;
			padding: 0;
		}
8.hover伪类选择器
		鼠标悬停时，调用内部css样式*/
		span:hover{
			color: white;
			background-color: red;
		}
		/*鼠标移入其父级，让子级的span修改样式
		注意：修改的样式必须是自己本身或后代*/

优先级权重值：
	1000  > 100> 10    > 1
	style > id > class > 标签名
	瘦死的骆驼比马大，高级永远大于低级
